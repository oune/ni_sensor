진동 측정 프로그램
1. vib.bat 파일을 더블클릭하여 실행
2. 측정 장치 선택
	메뉴에 나온 장치중 원하는 장치의 번호 입력
	ex)cDAQ2Mod4가 해당하는 숫자, 2입력
3. 채널 선택
	원하는 채널을 입력
	여러개 선택시 ex) ai0:3
	한개 선택시 ex) ai2

4. 파일이름 지정
	폴더명/파일이름 으로 경로 지정가능
	ex) data/aaa
	
이후 측정이 시작됨

온도 측정 프로그램
1. temp.bat 파일을 더블클릭하여 실행
2. 측정 장치 선택
	메뉴에 나온 장치중 원하는 장치의 번호 입력
	ex)cDAQ2Mod3가 해당하는 숫자, 1입력
3. 채널 선택
	원하는 채널을 입력
	여러개 선택시 ex) ai0:3
	한개 선택시 ex) ai2

4. 파일이름 지정
	폴더명/파일이름 으로 경로 지정가능
	ex) data/aaa
	
이후 측정이 시작됨

경로명 예시 
C:/Users/user250/Desktop/test/aaa
입력 하면 바탕화면 test 폴더에 aaa 파일들이 생성됨
